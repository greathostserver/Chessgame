# بازی شطرنج کامل

یک پیاده‌سازی کامل و کاربردی از بازی شطرنج که به طور خاص برای اجرا در GitHub Pages طراحی شده است.

## ویژگی‌ها

- ✅ تخته شطرنج ۸×۸ استاندارد
- ✅ همه مهره‌ها با حرکات صحیح
- ✅ تشخیص نوبت حرکات (سفید و سیاه)
- ✅ تاریخچه حرکات
- ✅ تایمر بازی
- ✅ قابلیت چرخش تخته
- ✅ پیشنهاد حرکت هوشمند
- ✅ قوانین کامل شطرنج
- ✅ رابط کاربری فارسی و زیبا
- ✅ واکنش‌گرا (Responsive) برای همه دستگاه‌ها

## نحوه استفاده

### اجرای محلی
1. همه فایل‌ها را در یک پوشه ذخیره کنید
2. فایل `index.html` را در مرورگر باز کنید
3. بازی به طور خودکار راه‌اندازی می‌شود

### استقرار در GitHub Pages
1. یک ریپازیتوری جدید در GitHub ایجاد کنید
2. همه فایل‌ها را آپلود کنید
3. به تنظیمات ریپازیتوری بروید
4. به بخش "Pages" بروید
5. در بخش "Source"، شاخه اصلی (main) را انتخاب کنید
6. روی "Save" کلیک کنید
7. بعد از چند دقیقه، بازی در آدرس `https://username.github.io/repository-name` قابل دسترسی خواهد بود

## قوانین بازی

این بازی شامل تمام قوانین استاندارد شطرنج است:

1. **حرکت مهره‌ها**: هر مهره با توجه به قوانین خاص خود حرکت می‌کند
2. **ضرب‌گیری**: مهره‌ها می‌توانند مهره‌های حریف را بگیرند
3. **کیش**: وقتی شاه مورد حمله قرار می‌گیرد
4. **کیش و مات**: وقتی شاه در کیش باشد و نتواند فرار کند
5. **پات**: وقتی هیچ حرکت قانونی وجود ندارد اما شاه در کیش نیست
6. **حرکات ویژه**: قلعه‌گیری، آنپاسان و ارتقای پیاده

## کنترل‌های بازی

- **بازی جدید**: شروع یک بازی جدید
- **بازگشت حرکت**: لغو آخرین حرکت (در این نسخه ساده، بازی را ریست می‌کند)
- **چرخش صفحه**: چرخاندن تخته برای دید بهتر
- **پیشنهاد حرکت**: دریافت یک حرکت پیشنهادی از سیستم
- **قوانین شطرنج**: مشاهده قوانین کامل بازی

## فناوری‌های استفاده شده

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- Font Awesome برای آیکون‌ها
- Google Fonts برای فونت فارسی

این پروژه به صورت ماژولار طراحی شده است و به راحتی می‌توان آن را توسعه داد:

1. **افزایش هوش مصنوعی**: بهبود موتور پیشنهاد حرکت

4. **تحلیل بازی**: اضافه کردن ابزارهای تحلیل حرکات
